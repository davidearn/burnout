## Package purpose

`burnout` is an R package for calculating the probability of epidemic burnout or persistence after a major outbreak.

The package is associated with:

```text
Parsons TL, Bolker BM, Dushoff J, Earn DJD (2024),
"The probability of epidemic burnout in the stochastic SIR model with vital dynamics,"
PNAS 121(5), DOI: 10.1073/pnas.2313708120.
```

The immediate development goal is to integrate creation of the conplot/contour-plot figure into the package using existing package functions where possible, rather than duplicating older standalone code.

## Conplot integration

For the immediate conplot integration, prefer adding new code in new files. Change existing files only when necessary to integrate, export, document, or test the new functionality.

The wrapper-level directory:

```text
burnout/sources/conplot_standalone/
```

contains messy historical standalone code used to create the original conplot/contour-plot figure. Treat it as read-only reference material. Do not edit, clean, or reorganize it unless explicitly asked.

Prefer using existing `burnout` package functions instead of duplicating functions from the standalone code. If a standalone function appears necessary because the package lacks equivalent functionality, report that clearly before copying or reimplementing it.

The immediate user need is slide-oriented conplot generation:

- ability to control contour-label size;
- ability to optionally include a colour-ramp legend on the right-hand side;
- ability to create the SIR/vital-dynamics conplot figure;
- ideally, ability to create the analogous SIRS conplot figure if existing package functions make that straightforward.
